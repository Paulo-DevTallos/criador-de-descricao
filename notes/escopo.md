<h2>Gerador de descrição ProDescriber</h2>
</hr>

# gerador de descrições
# funcionalidades padrão (logoff)
* será possível criar uma descrição
* será possível copiar uma descrição
* será possível gerar uma nova descrição
* será possível inserir um novo produto

# outras funcionalidades:
- criar topicos de copywriter relacionado aos produtos
- criar nuvem de palavras com palavras que engajam o produto na rede social
- criar tags / etiquetas que marquem clientes relacionado ao nicho ou tópicos dos produtos
- essas tags / etiquetas devem ter nome e descrição
